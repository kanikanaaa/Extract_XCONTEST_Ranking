Update Xcontest Rankings on DCZO website

Longest flights from Scheidegg

1. Visit the following Xcontest page. Xcontest login required: Link to the flights from Scheidegg in 2024
2. Show page source (right click -> view page source). Save this as html with the filename 'ranking.html'.
3. Start the Python script 'ranking' in the same directory. Copy the finished HTML into the DCZO website, see screenshot/

Japan ranking - open

1. View the page source of the following page (no login required): Link Japan Ranking-open
2. Save page source as html file with the name 'national_ranking.html'.
3. Start Python script 'club_ranking' in the same directory. -> copy the finished html into the DCZO website.
